Class name: AddExpense
Parent class: Actions
Children: none
Properties to note: none

Responsibilities:
* Adds a new expense from an existing category into a User's current
  budget (if they have one)
  Note: Does not put it down in their Transaction history nor does it
  adjust their balance; User must call for Withdrawal action separately to
  have it down on their record
   Variables:
	* Owner owner
    * String category
    * double amount

Collaborators:
* ActionResponse;
* AddExpenseRequest;
* AddExpenseResponse;
* Budget;
* Owner;
* OwnerRepository;
* ArrayList;
